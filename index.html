Project Roll

Below are some of the projects I have completed recently. The code is freely available on my GitHub Profile and they serve as a way of giving back to the open source community. Together, they have over 400 GitHub stars.

Beautiful PDFs from HTML

Date Published: April 4, 2021

In this popular guide (which trended to the top of HN and received over 150,000 reads), I walk through the basic steps of creating well-formatted PDFs from HTML & CSS, using the Paged.Js JavaScript library as a polyfill. I believe this approach is very useful, because the combination of HTML and CSS allows you to programmatically create beautifully typeset content without reliance on MS Word or other packages.

Link: https://pdf.math.dev

Online Math Editor

Date Published: March 1, 2021

This is an interesting project that combines a high quality rich text editor for the web (TinyMCE) with the power of MathJax.

Link: https://writer.math.dev

Common Lisp by Example

Date Published: November 21, 2020

In 2020 I began to learn the Common Lisp programming language. To improve my own understanding and serve as a reference for later, I started compiling a set of code examples, which culminated in a 52 page guidebook on the language. It has over 150 stars on GitHub and is well received. I personally don't refer to it anymore, but if you want to learn Lisp, I recommend it!

Link: Common Lisp by Example GitHub Repo

RPN 30

Date Published: February 4, 2019

Where it all began. As a long time fan of the HP 12C, I wrote a high quality RPN calculator for the iPhone, taking design cues from Apple's stock calculator, with some unique twists. It has over 10,000 downloads and rated 4.8 / 5 stars on the App store, with many reviews. This project was my first step back into programming after a decade of absence.

Link: Apple App Store Link
